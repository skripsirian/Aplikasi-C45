# Streamlit Application

This is a simple Streamlit application that demonstrates various Streamlit features including charts, tables, and statistics visualization.

## Setup

1. Install the required dependencies:
```bash
pip install -r requirements.txt
```

2. Run the Streamlit application:
```bash
streamlit run app.py
```

## Features

- Interactive sidebar with visualization options
- Line chart visualization
- Data table view
- Statistical analysis
- Responsive layout

## Requirements

See `requirements.txt` for a list of Python dependencies.
